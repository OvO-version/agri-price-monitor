## 智慧农情·丰收视界 —— 农产品价格监控系统

### 1. 项目简介  
本项目是一个面向农业市场的全栈价格监控与分析平台，旨在解决农产品价格信息分散、更新滞后的问题。系统通过定时爬取新发地批发市场官网的实时价格数据，经清洗后存入 MySQL 数据库，并提供丰富的可视化图表与智能分析功能，帮助用户快速掌握价格趋势、异常波动及采购时机。

### 2. 系统功能  
- **数据自动采集**：每日凌晨自动抓取新发地官网价格，支持手动一键触发更新。  
- **价格仪表盘**：展示总产品数、今日均价、最高价品种等关键指标。  
- **价格排行榜**：以横向柱状图展示当前页均价前 10 名的品种。  
- **最新报价表格**：分页显示当日各品种价格，支持按品名、产地、价格排序。  
- **价格异常预警**：自动计算每个品种今日价格与过去 7 日均价的涨跌幅，超过阈值时以脉冲圆点+文字提示。  
- **最佳买入推荐**：综合价格、波动率、季节性因子，量化得分后生成前 5 名购买建议。  
- **自定义趋势分析**：输入品名并选择近一月/近一年，生成平滑折线图并附 AI 文本分析。  
- **季节性价格热力图**：展示热门产品全年各月均价，颜色越红价格越高，直观反映旺季淡季。  
- **用户认证**：基于 Flask Session 的登录/登出，默认账号 admin/123456。  

### 3. 技术架构  

| 层级 | 技术栈 |
|------|--------|
| 后端框架 | Python Flask 2.3.3 |
| ORM | SQLAlchemy 1.4.48 + Flask-SQLAlchemy 2.5.1 |
| 数据库 | MySQL 8.0 (InnoDB，已建立复合索引) |
| 爬虫 | requests + BeautifulSoup4，增量更新与去重 |
| 定时任务 | APScheduler 3.10.4 (每日凌晨 1:00) |
| 前端 | HTML5 + CSS3 + JavaScript (ES6) |
| 可视化 | ECharts 5.5.0 (折线图、柱状图、热力图) |
| 图标库 | Font Awesome 6.0 |
| 其他 | pandas (数据清洗)、threading (后台爬虫)、logging (日志) |

### 5. 部署与运行  

#### 环境要求  
- Python 3.8+  
- MySQL 8.0  
- 已安装 pip 并配置虚拟环境（推荐）  

#### 安装依赖  
```bash
pip install flask flask-sqlalchemy sqlalchemy pymysql pandas requests beautifulsoup4 apscheduler
```

#### 数据库初始化  
1. 创建数据库 `agri_price`（字符集 utf8mb4）。  
2. 执行 `sql_init.sql`（项目文档中已提供）创建表及索引。  
3. 可选：运行 `main.py` 导入历史 CSV 数据。  

#### 启动应用  
```bash
python app.py
```
访问 `http://127.0.0.1:5000`，使用 `admin` / `123456` 登录（可以自己改）。

### 6. 使用说明  

- **登录后**进入仪表盘，左侧为价格排行榜与市场亮点卡片，右侧为预警、最佳买入及最新报价表格。  
- **表格**支持点击表头排序，底部提供分页。  
- **趋势分析**：在输入框中填写品名，选择时间范围，点击“生成”即可查看折线图及 AI 文本。  
- **季节性热力图**：自动展示热门产品各月均价，鼠标悬停显示具体数值。  
- **一键更新**：点击右上角“获取最新数据”，系统后台抓取当日新发地数据，等待数秒后自动刷新所有模块。  

### 7. 总结  
本项目实现了从数据采集、存储到可视化分析的全链路闭环，具备以下特点：  
- **自动化**：爬虫定时运行，用户无需手动导入数据。  
- **智能化**：基于历史数据计算价格异常和买入得分，辅助决策。  
- **可视化**：采用 ECharts 提供丰富的图表形式，直观呈现价格规律。  
- **易用性**：界面清新自然，操作流畅，响应迅速。
